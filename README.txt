##Working Locally
## Github automatically compares the code between the source warehouse and its own remote warehouse to see if there is a conflict. If there is a conflict, it will be displayed. Displaying Able to merge means that it can be merged. 
## Local library and remote library The local library is when the developer submits the file to the Git warehouse, and the Git warehouse file is still on the local computer. The GitHub remote warehouse can be managed through the Git warehouse.
